# 05 -- Cinematic Hero

## Design Philosophy

Full-screen hero sections, parallax depth, and atmospheric effects. The visuals ARE the message. This template treats every viewport as a frame in a film -- minimal text, maximum visual impact. The design does not explain; it immerses. If the client's visuals are strong, this template amplifies them. If the visuals are weak, this template exposes that immediately.

## UX Pattern

The user lands on a full-viewport cinematic hero with atmospheric effects -- floating embers, a subtle vignette, and a slow kenBurns zoom on the background image. Text is minimal and overlaid on dark visuals. As the user scrolls, parallax layers create depth: backgrounds move slower than foreground content, and sections crossfade between one another.

A full-bleed media section follows with no text overlay -- pure visual. Features appear in a muted 2-column layout with large icons and short descriptions, kept brief because the images do the heavy lifting. A single dramatic testimonial in large type against a dark background adds social proof with theatrical confidence. The page closes with an accent CTA.

Five sections total: cinematic hero, full-bleed media, understated features, dramatic single testimonial, and accent CTA.

## When to Use

- Photography studios or portfolios where the images need to dominate
- Film, entertainment, or event promotion sites
- Hospitality brands -- hotels, resorts, restaurants -- that sell through atmosphere
- Real estate or property showcases where visuals close the deal
- Any brand with stunning visual assets that deserve full-screen treatment

## When NOT to Use

- Brands with weak or generic photography -- this template will magnify the problem
- Content-heavy products that need to communicate detailed features or specifications
- SaaS or developer tools where technical information matters more than mood
- Clients who need fast page loads on low-bandwidth connections -- parallax and large media are expensive
- Accessibility-critical projects where motion effects may cause issues for users with vestibular sensitivities
